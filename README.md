# CursoReact2025 - eCommerce React

## Alcance:
* Carrito de compras
* Autenticación con LocalStorage
* CRUD completo de productos
* Búsqueda y paginación
* Diseño responsive
* Animaciones y notificaciones

## Tecnologías
Este proyecto es un eCommerce desarrollado con React 19 + Vite, utilizando:

* Las librerias React Router DOM para navegación, React Toastify, React Icons y Framer Motion para mejorar la experiencia de usuario. 
* Las librerias Bootstrap + styled-components para un diseño moderno y responsive.
* Context API para el carrito y la autenticación.
* MockAPI para el CRUD de productos.


## Instalación
### Clonar el repositorio
```bash
git clone https://github.com/FiorellaA17/CursoReact2025.git
```
### Entrar a la carpeta del proyecto
```bash
cd CursoReact2025
```
### Instalar dependencias
```bash
npm install
```
### Ejecutar la aplicación
Para iniciar el entorno de desarrollo:
```bash
npm run dev
```
Vite generará una URL local, usualmente:
```bash
http://localhost:5173
```
